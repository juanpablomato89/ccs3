# CSS3 — CSS3 Animations & Effects Examples

A collection of **CSS3** animation and visual effect examples demonstrating transitions, keyframes, transforms, and modern layout techniques.

## Topics Covered

- CSS transitions and hover effects
- `@keyframes` animations
- CSS transforms (rotate, scale, translate, skew)
- Flexbox and Grid layouts
- CSS variables and custom properties
- Pseudo-elements and pseudo-classes
- Responsive design with media queries

## Purpose

Reference collection and practice ground for advanced CSS3 techniques used in production Angular/HTML projects.
